# 📖 Overview

MovieHub is a full-stack web application built using Django that provides an online movie ticket booking experience.

The application allows users to register, log in securely, browse theatres, explore available movies and show timings, and reserve seats through a database-driven booking workflow.

This project demonstrates backend development using Django, relational database design, user authentication, template inheritance, and deployment on Render.

# 🚀 Getting Started

## Clone Repository

```bash
git clone https://github.com/Mytresh-gowd/movie-ticket-booking-django.git
```

Move into project

```bash
cd movie-ticket-booking-django
```

Create Virtual Environment

```bash
python -m venv mb_env
```

Activate Environment

### Windows

```bash
mb_env\Scripts\activate
```

### Linux / macOS

```bash
source mb_env/bin/activate
```

Install Dependencies

```bash
pip install -r requirements.txt
```

Apply Migrations

```bash
python manage.py migrate
```

Run Server

```bash
python manage.py runserver
```

Open

```
http://127.0.0.1:8000/
```
